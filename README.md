## Hotline Jukebox

The Hotline Jukebox is my project playing with music, analog technology, storytelling, and a bit of electrical engineering built for my class "Creative Technology" taught by Zack Weaver at the ATLAS Institute at the University of Colorado Boulder. Based around an old telephone and an Arduino Uno, Hotline Jukebox bridges the gap between technology, story, and environmental storytelling. 

You can find more information about the build on my [website](https://www.andrewpatra.com/work/hotline).

## Using the Project

If you'd like to replicate this project, all you need to do is supply your own mp3 tracks. You can find how to set up the file system for the DF Player [Here](https://wiki.dfrobot.com/dfr0299/#docs).
